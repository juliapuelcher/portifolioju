Projetos

Aplicativo Interativo de Química Orgânica em Flutter

Este aplicativo Flutter foi desenvolvido com o objetivo de ensinar conceitos fundamentais de Química Orgânica de forma visual, acessível e interativa. É ideal para estudantes do ensino médio que estão se preparando para vestibulares e o ENEM.

Projeto de TCC | Curso Técnico do SENAI

O Nutrigestão é um aplicativo mobile desenvolvido com Flutter como parte do meu Trabalho de Conclusão de Curso (TCC) no SENAI. O app foi criado com o objetivo de organizar, visualizar e acompanhar cardápios escolares de forma prática e acessível, tanto para alunos quanto para responsáveis e equipe escolar.

Raízes da Luta – Projeto Educacional Antirracista

O Raízes da Luta é um site educativo desenvolvido em HTML e CSS como parte de um projeto de linguagem de programação no ensino médio. Inspirado no livro "Pequeno Manual Antirracista", da filósofa Djamila Ribeiro, o site busca fomentar a reflexão e o combate ao racismo estrutural por meio de conteúdos acessíveis, interativos e didáticos.

Projeto Interdisciplinar de Química e Programação Web

Reação PHácil é um site didático desenvolvido em HTML e CSS, criado como parte de um projeto educacional para aprofundar os conhecimentos sobre funções inorgânicas na disciplina de Química. O site tem como foco explicar de forma clara e visual os principais conteúdos sobre ácidos, bases, sais e pH — tópicos fundamentais no ensino médio.
